Bluehaven 🌊
A full-stack digital wellbeing web application built with the MERN stack.
Bluehaven allows users to track habits, log weekly moods, and visualize personal insights through interactive analytics.
🔗 Live Demo:
Frontend: https://bluehavenn.netlify.app⁠
Backend: https://bluehaven-backend.onrender.com
🔗 Features
JWT Authentication (Login / Signup)
Protected Routes (Frontend + Backend)
Habit Tracker (Full CRUD)
Mood Log (Emoji-based weekly tracking)
Insight Dashboard (Habit & Mood analytics)
Feedback / Support submission
Reusable Modal Component
Deployed (Netlify + Render)
🔗 Tech Stack
Frontend:
React (Vite), 
React Router,
Chart.js
Backend:
Node.js,
Express,
MongoDB Atlas,
JWT, 
bcrypt
Deployment:
Netlify + Render
🔗 Architecture Highlights
Modular backend structure (controllers, routes, middleware)
Secure authentication using JWT
Environment-based API configuration
SPA routing configured for production deployment
Separated frontend and backend services
