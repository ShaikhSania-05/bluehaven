Bluehaven 🌊

Bluehaven is a frontend web project built using React and Vite.
The project focuses on a clean UI, modular components, and a structured layout.

## Tech Stack

- React
- Vite
- JavaScript
- CSS

## Project Structure

- `src/components` – UI components
- `src/styles` – custom styling
- `public` – static assets
